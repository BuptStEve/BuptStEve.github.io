## How to play?
* Use `WASD` to move
* Tap `Space` to jump
* Double tap `Space` to fly, tap `Shift` to get down
* Tap `R` to toggle view(First Preson/Third Person)

## How to run?
```bash
$ npm start
# OR
$ yarn start
```
